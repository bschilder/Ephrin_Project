========================================
NIH Blueprint Non-Human Primate (NHP) Atlas
========================================

This archive contains expression data for rows 1 through 3 of a 'search' search
at a 'SAMPLES' resolution level, arranged by age,structures.

The file Expression.csv contains expression values, calculated using zscore normalization.
Each row begins with the ID of a probe.

The file Columns.csv contains metadata for each column in Expression.csv, arranged in the same order.

The file Probes.csv contains metadata for the probes referenced in Expression.csv.

The original search can be reproduced on our web site at http://www.blueprintnhpatlas.org/microarray/search/show?exact_match=false&search_term=EPHA7&search_type=gene
