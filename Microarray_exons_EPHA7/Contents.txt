=========================
BrainSpan: Atlas of the Developing Human Brain
=========================

This archive contains expression data for rows 1 through 20 of a 'search' search arranged by structures,donors

The file Expression.csv contains log2 expression levels.

Each row begins with the ID of a probe.

The file Columns.csv contains metadata for each column in Expression.csv, arranged in the same order.

The file Rows.csv contains metadata for the probes referenced in Expression.csv.

The original search can be reproduced on our web site at http://www.brainspan.org/rnaseq/gene/1093963
